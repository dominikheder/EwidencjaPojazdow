# EwidencjaPojazdow
Projekt składa się z dwóch części realizowanych w ramach Etapów 1-4:

### 1. Aplikacja Desktopowa (WinForms)
- Zarządzanie listą pojazdów w DataGridView.
- Raportowanie RDLC (pojazdy do przeglądu w tym miesiącu).
- Eksport do PDF.

### 2. Interfejs Webowy (Bootstrap 5)
- Responsywna tabela pojazdów.
- Formularz dodawania w oknie Modal.
- Pełna walidacja pól za pomocą JavaScript.
